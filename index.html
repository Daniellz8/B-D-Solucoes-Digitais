Um projeto de exemplo para praticar controle de versão

O que é Git?

Git é um sistema de controle de versão distribuído, o que significa que ele rastreia as alterações em qualquer conjunto de arquivos. É a ferramenta que usamos para gerenciar o histórico do nosso código.

Como Usar?

Vamos adicionar um novo recurso para praticar. Siga estes passos simples no terminal:

  • 1. Faça uma alteração nesta página (por exemplo, mude um texto).
  • 2. Salve o arquivo.
  • 3. Abra o terminal na pasta do projeto.
  • 4. Use `git add .` para preparar suas alterações.
  • 5. Use `git commit -m "Adicionando novo recurso"` para salvar a versão.
  • 6. Use `git push` para enviar para o GitHub.

Seção de Exemplo

Esta seção foi criada para vocês praticarem a colaboração. Seu amigo pode fazer uma alteração aqui, enviar para o GitHub e você pode usar o comando `git pull` para obter as alterações dele.
